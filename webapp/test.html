Functional programming against the DOM with React

This is a RESTful CRUD editor driven by a JSON schema written in a functional style. The code has the following properties:

  • no Backbone models
  • no Knockout viewmodels
  • all view rendering code is stateless
  • referentially transparent programming model - no mutable state at all, unless you choose it
  • imperative-style widget library (with mutable state) wrapped into functional-style programming model

The intent of this page is to go along with a readthrough of the source code.

If you're here from rawgithub.com, the javascript can take up to 15 seconds to load.
